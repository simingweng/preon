The package providing a default implementation of the builder interfaces. The implementation in this package tries to ease construction of packages constructing output by defining a LifecycleListener that gets notified when new document elements are pushed on the stack, or when these document elements are popped from the stack.

Note that the LifecycleListener will not always be called directly when a new document element is constructed. Take for instance a section. Whenever you would create a new section by invoking .section("foobar") on the current element, you would still be allowed to pass some additional metadata. The LifecycleListener operation is invoked whenever you are done with passing additional metadata. (In case of a section, that would typically be when you start to add content to the section.)
